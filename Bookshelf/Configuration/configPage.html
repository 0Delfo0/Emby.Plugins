Before MB Bookshelf can recognize your books, a collection needs to be added to either the Default Media Library or a users media library with the collection type set to Books. MB Bookshelf will do the rest

Supported e-book formats

  • mobi
  • epub
  • pdf
  • cbr
  • cbz

MediaBrowser Clients

MediaBrowser Classic

In order to view books on MBC you will need to install the client plugin for MB Bookshelf. It is available for download from the MB Classic configurator. Once installed there is no configuration necessary, just click your book and the plugin will pass the request to the operating system. You will need to have an appropriate viewer installed. Users are free to use their favorite programs but I recommend Calibre for epub/mobi & ComicRack for cbz/cbr.

MediaBrowser Android

There is no required setup for this client. As above, the book will be passed to the operating system to open. You will need appropriate viewers on the device. As Android doesn’t natively see Windows shares, the books are downloaded to the devices download folder before being opened.

MediaBrowser Theater

To view books with MBT, you will need to add at least one Custom Player. The player needs to be set to External Player, and the MediaType specified as Book. Select the file extensions this player will open and then provide the path to the executable used to view the books Note: as MBT is still in development, these directions may change
